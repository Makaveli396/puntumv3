📦 BOT DE TELEGRAM - CINEBOT

🔧 REQUISITOS:
- Python 3.10 o superior
- pip install -r requirements.txt

📄 CONFIGURACIÓN:
1. Copia el archivo `.env.example` como `.env`
2. Rellena tus datos:
   - BOT_TOKEN= (lo obtienes de BotFather)
   - ADMIN_IDS= tu ID de Telegram
   - TMDB_API_KEY= clave de TheMovieDB (https://www.themoviedb.org/)

▶️ PARA EJECUTAR:
python bot.py

✅ Las tablas de la base de datos se crean automáticamente.

🛠 SOPORTE:
Si tienes problemas, revisa los logs en consola.
